/* Jasmine Matcher's -> Which implements the boolean comparison between Actual and Expected Value */

1. toBe()
2. toEqual()
3. toBeTruthy()
4. toBeFalsy()
5. not -> negating the matcher -> we can use with any matcher
6. toBeUndefined()
7. toBeDefined()
8. toBeNull()
9. toContain()
10. toBeNaN()
11. toThrow()
12. toThrowError()
13. toMatch() -> to matching the actual data using regular expression

/* Asymmetric Matcher - Jasmine.anything */
1. jasmine.anything() - used with toEqual(), toContain() and toHaveBeenCalledWith() - It should not null or undefiend
2. jasmine.any() - To check the instance of the class or constructor
3. jasmine.objectContaining()
4. jasmine.stringContaining()
